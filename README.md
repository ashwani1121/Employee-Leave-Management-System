Employee Leave Management System (ELMS)

The Employee Leave Management System helps companies manage employee leave in a simple and organized way.
This project allows employees to apply for leave online, and managers/admins can review, approve, or reject requests easily.

What this project does
Employees can apply for different types of leave (Sick Leave, Casual Leave, etc.)
Managers can approve or reject leave requests.
Admin can manage employees, leave types, and overall system settings.
Users can track their leave balance and leave history.
Secure login system to make sure only authorized people can access the platform.
Real-time status updates for every leave request.

Technologies Used
Backend: Spring Boot (Java)
Database: MySQL / JDBC / JPA
Frontend:HTML / CSS / JavaScript / [React.js (Optional if needed)]
Authentication: JWT (JSON Web Token)

Tools: Maven, Postman, Git & GitHub

Key Features
Simple and user-friendly UI

Role-based access: Admin, Manager, Employee

REST APIs for applying and approving leaves
Leave balance calculation
Email or notification support (optional)
Easy to deploy on servers like AWS

Why this project?
This system removes manual leave tracking using paper or Excel sheets.
It helps keep everything organized, reduces errors, and saves time for HR teams and employees.

How to run this project
Clone the repository
Import into IDE (IntelliJ/Eclipse)
Configure MySQL database
Run the Spring Boot project
Use Postman or frontend to test APIs
